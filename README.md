## üöÄ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/student-portfolio.git
cd student-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

### Build for Production
```bash
npm run build
```

## üìù Customization

### Personal Information
Update the following files with your information:
- `src/pages/Home.js` - Name, title, and introduction
- `src/pages/About.js` - Education and background
- `src/pages/Resume.js` - Resume content
- `src/components/Footer.js` - Social links

### Projects
Edit `src/pages/Projects.js` to add your own projects:
- Project titles and descriptions
- Technologies used
- GitHub and demo links
- Project features

### Skills
Modify `src/pages/Skills.js` to reflect your skill set:
- Programming languages
- Frameworks and libraries
- Tools and technologies

### Contact Information
Update contact details in:
- `src/pages/Contact.js`
- `src/components/Footer.js`
